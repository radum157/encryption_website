Homepage - what is this website used for?

This site provides access to a basic Steganography tool. You can choose to encrypt a message inside of a given image, or to decrypt an image, printng out the secret message it contains. The least significant bits algorithm (LSB) is used on the cryptography part. See this for more information on the website and the motivation behind it. Please use one of the supported image formats (.png or .bmp, since they are lossless when compressing).
